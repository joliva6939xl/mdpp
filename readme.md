# MDPP - Partes Virtuales

Proyecto dividido en 2 partes principales:

- `Movil/` → App móvil (Expo + React Native + Expo Router)
- `api/` → Backend (Node.js + Express + PostgreSQL)

---

## 1. REQUISITOS EN CADA PC NUEVA

Antes de ejecutar el proyecto en una PC nueva, necesitas:

1. Node.js y npm instalados  
2. PostgreSQL instalado  
3. Git (para clonar el repositorio)
